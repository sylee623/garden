A flexible hippocampal population code for experience relative to reward

연구 배경 및 목적 🎯

보상은 동물의 행동을 유도하는 핵심 사건이며, 이 전후의 사건들을 기억하는 것은 학습에 필수적이다.
Hippocampus는 spatial 및 non-spatial modalities (e.g. time, multisensory input, task variables)에 걸쳐 behavioral timescale의 시퀀스를 생성하지만, reward를 중심 기준으로 전체 experience를 재구성하는 **reward-anchored population code**가 존재하는지는 알려지지 않았다.
본 연구는 마우스가 virtual linear track에서 보상 위치를 학습하고 재학습하는 과정에서 CA1 뉴런들이 reward-relative remapping을 수행하는지, 그리고 그 representation이 behavioral adaptation과 어떻게 연결되는지를 규명하고자 했다.

주요 발견 (Highlights) 🌟

  1. Reward-relative (RR) place cells는 reward 위치가 변경되면 field를 같은 reward의 상대적인 위치로 재배치함 (even hundreds of cm away).
  2. RR 뉴런의 firing 시퀀스는 보상 위치 변경 후에도 **firing order**를 유지하며, behavioral timescale sequences를 형성.
  3. 학습이 진행될수록 RR representation이 강화되며, anticipatory licking 정밀도와 RR 시퀀스 정밀도가 정량적으로 연결됨.
  4. TR (track-relative) 뉴런은 spatial location을 기준으로 안정적으로 firing하며, RR과 병렬적 population으로 존재.
  5. RR 뉴런의 remapping은 licking/speed 등의 행동 변화보다 선행되며, reward expectation update를 반영할 가능성 있음.

사용된 방법 요약 🔬

실험 설계

  • Head-fixed mouse가 3가지 hidden reward zone 중 하나를 가진 virtual linear track(450cm)을 반복 주행
  • Reward zone은 task day별로 위치가 변경되며(7일간 switch), 2개의 visually distinct environments로 교차

Imaging 및 분석

  • Two-photon calcium imaging (GCaMP7f in dorsal CA1)
  • Reward-relative (RR) coordinate system 기반 remapping 분석
  • Spatial tuning, population vector correlation, GLM modeling, decoding, sequence correlation, remap timing 등 정량 분석
  • Shuffling, circular statistics, cross-validation, time-warping, teleport zone analysis 포함 ,

    Figure/소챕터별 요약 📊

    1. Monitoring neural activity during a reward learning task

  • 마우스는 hidden reward zone 위치를 빠르게 학습하며, anticipatory licking과 speed 감소가 나타남
  • Session마다 보상 위치를 바꾸거나 환경을 변경하여 remapping 분석 유도

2. Moving reward induced remapping spanning the environment

  • Place cell의 약 16%가 RR remapping을 보여줌 (보상과 ±80cm 이상 떨어진 위치에서도)
  • 이러한 remapping은 보상 이동이 spatial cue가 아니라 **reward memory**에 의해 유도된다는 것을 의미

3. RR population provides accurate decoding of RR position

  • RR subpopulation만이 reward 위치가 바뀐 후에도 **reward-centered position**을 정확히 decode
  • TR이나 non-RR remapping subpopulation은 post-switch trial에서는 decode 정확도가 떨어짐

4. Behavioral timescale sequences relative to reward

  • ==RR 뉴런들의 firing order는 reward switch 후에도 유지됨 → experience-relative firing sequences ==
  • RR 시퀀스는 보상 시작 위치 이후 영역에 과도하게 분포하며, 학습일수에 따라 점차 precision이 증가
  • TR 시퀀스는 track boundary 근처에 몰려 있음

5. Dynamic cell recruitment into the RR population

  • RR 뉴런들은 day-to-day로 유동적이며, 처음엔 TR 또는 non-place cell이었던 뉴런들도 RR로 reclassification됨
  • 반면 TR 뉴런은 구성원이 고정되지 않고 지속적으로 turnover 됨 → representational drift

6. RR cell activity often updates before behavior

  • RR remapping이 licking 또는 running speed 변화보다 평균 1~2 trial 앞서 발생
  • 이는 RR 코딩이 reward prediction error 또는 internal reward model update와 연결됨을 시사

Discussion 요약 💬

  • 해마는 spatial map (TR)과 reward-relative map (RR)을 병렬적인 population code를 통해 동시에 유지함
  • RR 코드는 단순한 reward location over-representation이 아니라, reward를 기준으로 한 전체 경험 시퀀스의 재배열
  • RR 뉴런의 구성은 고정된 세포 subtype이 아닌 유연하게 재구성되는 ensemble이며, behavioral performance와 함께 정교화됨
  • RR remapping은 action execution보다 앞서 발생하며, 해마가 행동 결정에 causal한 역할을 할 수 있음을 시사
  • 기존 spatial reward studies와 달리, 본 연구는 보상을 spatial cue로부터 dissociate시켜 RR 코딩의 순수성을 확보

Related Article

Abstract

[!summary] Abstract
To reinforce rewarding behaviors, events leading up to and following rewards must be remembered. Hippocampal place cell activity spans spatial and non-spatial episodes, but whether hippocampal activity encodes entire sequences of events relative to reward is unknown. Here, to test this possibility, we performed two-photon imaging of hippocampal CA1 as mice navigated virtual environments with changing hidden reward locations. We found that when the reward moved, a subpopulation of neurons updated their firing fields to the same relative position with respect to reward, constructing behavioral timescale sequences spanning the entire task. Over learning, this reward-relative representation became more robust as additional neurons were recruited, and changes in reward-relative firing often preceded behavioral adaptations following reward relocation. Concurrently, the spatial environment code was maintained through a parallel, dynamic subpopulation rather than through dedicated cell classes. These findings reveal how hippocampal ensembles flexibly encode multiple aspects of experience while amplifying behaviorally relevant information.
